# Wstęp do programowania

To start learning **C++** you'll have to use a *code editor*.

## Wybór edytora kodu

Zalecamy użycie edytora kodu online przez kilka pierwszych lekcji. Będzie to dla Ciebie proste i nie będzie wymagało żadnej dodatkowej konfiguracji.

Polecane edytory kodu online:

- [Repl.it](http://repl.it)

## Tworzenie swojej pierwszej aplikacji

Open a code editor, then type the following code

```cpp
#include <iostream>

int main()
{
	std::cout << "Hello, world!";
}
```

Congratulations! You've printed your first `Hello, World!`
