Aby programować w języku C++ potrzebujesz przynajmniej dwóch narzędzi:

- edytora kodu
- kompilatora

W skrócie:

- **edytor kodu** pozwala nam... edytować kod programu, zarządzać strukturą plików w projekcie
i zwykle posiada też wiele innych przydatnych funkcji
- **kompilator** tłumaczy kod programu na format zrozumiały dla komputera, <u>tylko</u> jeśli kod jest poprawny
tj. napisany zgodnie ze składnią języka C++

Połączenie tych dwóch rzeczy w jedno narzędzie jest nazywane **środowiskiem** (ang.: *IDE - integrated development environment*).

## 🚀 Zacznij od
